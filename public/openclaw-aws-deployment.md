## はじめに

「AIエージェントを使いたいけど、Mac miniを買うのはちょっと…」

そんな悩みを解決する方法があります。**OpenClaw（旧Clawdbot AI）をAWSで動かす**という選択肢です。

この記事では、Mac miniを買わずに月額約300円でAIエージェントを24時間動かす方法を紹介します。

## この記事でできるようになること

- Mac miniなしでOpenClawを動かす
- 月額約300円（$2程度）で24時間稼働
- 10分で環境構築完了

## なぜAWSで動かすの？

「自宅のMacで動かせばいいのでは？」と思うかもしれません。

でも、クラウドで動かすメリットは大きいんです。

### ローカル vs クラウドの比較

| 観点 | ローカル（Mac mini等） | クラウド（AWS） |
|------|---------------------|----------------|
| 初期費用 | 10万円以上 | 0円 |
| 月額費用 | 電気代 | 約300円 |
| アクセス | 自宅のみ | どこからでも |
| 稼働時間 | PC起動中のみ | 24時間365日 |
| セキュリティ | 自宅ネットワーク | VPCで分離 |

**「試しに使ってみたい」という段階なら、AWSの方がハードルが低いです。**

## 構築手順（簡単3ステップ）

### Step 1: AWSにデプロイする（約5分）

CloudFormationテンプレートを使います。

1. AWSマネジメントコンソールにログイン
2. CloudFormationでスタックを作成
3. テンプレートをアップロード
4. 約1〜3分待つだけ

これでEC2インスタンスが自動的に作成されます。

### Step 2: OpenClawの初期設定

作成されたEC2インスタンスにSSH接続します。

```bash
ssh -i your-key.pem ec2-user@<インスタンスのIPアドレス>
```

接続したら、セットアップコマンドを実行します。

```bash
openclaw_setup
```

いくつか質問されるので、以下のように選択します。

| 質問 | 選択 | 説明 |
|-----|-----|-----|
| Operation Mode | `Local` | ローカルモードで動作 |
| LLM Provider | `OpenRouter` | 無料モデルが使える |
| Model | `gpt-oss-120b:free` | 無料で使えるモデル |

### Step 3: 動作確認

セットアップが完了すると、URLが表示されます。

そのURLにブラウザでアクセスすれば、OpenClawと対話できます！

## 気になるコスト

### EC2インスタンスの費用

| 利用パターン | 費用 |
|-------------|------|
| 1日8時間稼働 | 約10円/日（約300円/月） |
| 24時間稼働 | 約600円/月 |

### LLMの費用

OpenRouterの無料モデル（`gpt-oss-120b:free`）を使えば**追加費用なし**です。

有料モデルを使う場合は、使用量に応じて別途費用がかかります。

## 注意点

:::note warn
この構成は検証・個人利用向けです。本番環境での利用には追加のセキュリティ対策が必要です。
:::

- セキュリティグループで必要なポートのみ開放する
- SSH鍵は適切に管理する
- 不要になったらスタックを削除してコストを抑える
