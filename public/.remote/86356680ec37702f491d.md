## はじめに

「AIエージェントを使ってみたいけど、高いMacを買わないといけないの？」

そんな悩みを解決する方法があります。OpenClaw（旧Clawdbot AI）をAmazon AWSで動かすことで、専用ハードウェアなしで月額約300円からAIエージェントを利用できます。

この記事では、AWSアカウントさえあれば誰でもできる、OpenClawのクラウドデプロイ方法を解説します。

## この記事でできるようになること

- Mac mini等の専用機なしでOpenClawを動かす
- どこからでもアクセスできるAIエージェント環境を構築
- 月額約300円（8時間/日利用の場合）で運用

## クラウドデプロイのメリット

| ローカル実行 | クラウドデプロイ |
|-------------|----------------|
| Mac mini等が必要 | ハードウェア不要 |
| 自宅からのみアクセス | どこからでもアクセス |
| 電気代がかかる | 使った分だけ課金 |
| PCをつけっぱなし | 24時間稼働可能 |

## 必要なもの

- AWSアカウント（無料で作成可能）
- LLMプロバイダーのAPIキー（OpenRouter推奨、無料枠あり）

## 手順

### ステップ1: AWSスタックのデプロイ（約5分）

1. AWSコンソールにログイン
2. CloudFormationを開く
3. 「スタックの作成」をクリック
4. 提供されているテンプレートをアップロード
5. スタック名を入力して「作成」

```bash
# CLIの場合
aws cloudformation create-stack \
  --stack-name openclaw-stack \
  --template-body file://template.yaml \
  --capabilities CAPABILITY_IAM
```

### ステップ2: 設定

EC2インスタンスにSSH接続して、以下を設定します。

```bash
# 動作モードの選択
# → ローカルを選択

# LLMプロバイダーの選択
# → OpenRouter（無料枠あり）を選択

# モデルの選択
# → gpt-oss-120b:free 等を選択
```

### ステップ3: アクセス

設定完了後、表示されるURLにアクセスすると、AIエージェントとの対話を開始できます。

## 料金の目安

### EC2の料金

| 使い方 | 月額（概算） |
|-------|------------|
| 1日8時間 | 約300円 |
| 1日24時間 | 約900円 |

### LLMの料金

OpenRouterの無料モデルを使う場合は、LLMの追加費用はかかりません。

## よくあるトラブルと解決方法

### スタック作成でエラーが出る

- IAM権限が不足している可能性があります
- `CAPABILITY_IAM`オプションを忘れていないか確認してください

### インスタンスに接続できない

- セキュリティグループでSSH（ポート22）が許可されているか確認
- キーペアが正しく設定されているか確認

### URLにアクセスできない

- セキュリティグループでHTTP/HTTPS（ポート80/443）が許可されているか確認
- インスタンスが起動中か確認

## コスト削減のコツ

### 使わない時は停止する

AWSコンソールからインスタンスを停止すれば、その間の料金はかかりません。

### スポットインスタンスを使う

最大90%オフで利用できますが、中断される可能性があります。お試し用途には最適です。
